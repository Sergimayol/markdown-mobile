# Markdown mobile

## What is it?

Markdown mobile is a mobile app for Android and iOS that allows you to preview Markdown files.

## Development

```bash
npm install
expo start
```

## Project structure

```bash
.
├── App.tsx        # Entry point
└── src/
    ├── screens/    # Screen components
    ├── components/ # UI components
    ├── utils/      # Custom hooks and helpers
    └── assets/     # Image files
```
